# Yogaverse Pro

Yogaverse Pro is a simple AI-powered website made using HTML and the Google Gemini API.  
This project was created for learning and experimenting with AI integration in a basic web interface.

## Features
- Clean and simple UI
- Uses Google Gemini API for AI responses
- Works on mobile and desktop
- Fully hosted on GitHub Pages

## How to Use
1. Open the website in any browser.
2. Type your message.
3. Click the send button to get an AI response.

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Gemini API
- GitHub Pages
